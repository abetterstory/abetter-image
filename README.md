ABetter Laravel Image is a package of image tools for faster development of component-based web applications, with focus on scalable static caching.

---

## Requirements

* PHP 7.3+
* Composer 1.6+
* Laravel 6.0+

---

## Installation

Via Composer:

```bash
composer require abetter/image
```
